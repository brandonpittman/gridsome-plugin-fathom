# gridsome-plugin-fathom

Quickly set up Fathom Analytics in your Gridsome project.

```sh
npm install gridsome-plugin-fathom
```

```javascript
// gridsome.config.js

plugins: [
{
  use: 'gridsome-plugin-fathom',
    options: {
      siteId: 'your-site-id',
      debug: false // set to true for local debugging; defaults to false
    }
  }
]
```
